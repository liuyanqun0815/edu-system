# 系统管理与配置需求文档

## 一、模块概述

**模块名称**：系统管理与配置  
**对应控制器**：SysConfigController、SysSettingController、SysActivityController、SysEvaluationController、SysExcelTemplateController、SysEmailTemplateController、SysWordController、SysParentController  
**数据库表**：sys_config_category、sys_config_item、sys_setting、sys_activity、sys_activity_role、sys_evaluation、sys_excel_template、sys_excel_template_column、sys_email_template、sys_email_log、sys_word、sys_parent、sys_user_parent

## 二、系统配置管理

### 2.1 二级配置体系

**架构设计**：
```
sys_config_category（配置分类）
  └── sys_config_item（配置枚举项）
  
sys_setting（系统全局设置）
```

### 2.2 配置分类管理

**数据库表**：sys_config_category

**预置分类**（10个）：
| ID | 分类名称 | 分类编码 |
|----|---------|---------|
| 1 | 系统名称 | system_name |
| 2 | 系统Logo | system_logo |
| 3 | 系统描述 | system_desc |
| 4 | SMTP配置 | smtp |
| 5 | 默认头像 | default_avatar |
| 6 | 密码有效期 | password_expiry |
| 7 | 登录锁定策略 | login_lock |
| 8 | 默认考试时长 | exam_duration |
| 9 | 及格分数线 | pass_score |
| 10 | 其他配置 | other |

**接口**：
```
GET    /system/config/category/list    # 获取所有分类
POST   /system/config/category         # 新增分类
PUT    /system/config/category         # 修改分类
DELETE /system/config/category/{id}    # 删除分类
```

### 2.3 配置项管理

**数据库表**：sys_config_item

**字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| categoryId | bigint | 分类ID |
| itemKey | varchar(100) | 配置项键 |
| itemValue | text | 配置项值 |
| description | varchar(200) | 描述 |

**预置配置项**（53项）：

**系统参数**：
- system_name：智慧教培管理系统
- system_logo：Logo URL
- system_desc：系统描述

**邮件参数**：
- smtp_host：SMTP服务器地址
- smtp_port：SMTP端口（25/465/587）
- smtp_username：邮箱账号
- smtp_password：邮箱密码
- smtp_from：发件人

**用户策略**：
- default_avatar：默认头像URL
- password_max_age：密码有效期（天）
- login_max_fail：登录失败最大次数
- login_lock_duration：锁定时长（分钟）

**考试参数**：
- default_exam_duration：默认考试时长（分钟）
- default_pass_score：默认及格分数线

**接口**：
```
GET    /system/config/item/list/{categoryId}    # 获取分类下配置项
POST   /system/config/item                       # 新增配置项
PUT    /system/config/item                       # 修改配置项
DELETE /system/config/item/{id}                  # 删除配置项
```

### 2.4 系统设置

**数据库表**：sys_setting

**字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| settingKey | varchar(100) | 设置键 |
| settingValue | text | 设置值 |
| description | varchar(200) | 描述 |

**分组管理**：
- basic：基础设置（系统名称、Logo等）
- notify：通知设置（邮件、短信等）

**接口**：
```
GET    /system/setting/list          # 获取所有设置
GET    /system/setting/group/{group} # 获取分组设置
PUT    /system/setting               # 修改设置
```

## 三、活动公告管理

### 3.1 活动管理

**功能描述**：发布系统公告、课程活动、考试活动等

**数据库表**：sys_activity

**活动类型**：
| 类型编码 | 类型名称 | 说明 |
|---------|---------|------|
| 1 | 系统公告 | 全站通知 |
| 2 | 课程活动 | 课程相关活动 |
| 3 | 考试活动 | 考试相关活动 |
| 4 | 用户活动 | 用户相关活动 |

**接口**：
```
GET    /system/activity/page           # 分页查询活动
GET    /system/activity/{id}           # 活动详情
POST   /system/activity                # 发布活动
PUT    /system/activity                # 修改活动
DELETE /system/activity/{id}           # 删除活动
POST   /system/activity/{id}/publish   # 发布活动
POST   /system/activity/{id}/revoke    # 撤回活动
```

**活动字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| title | varchar(200) | 活动标题 |
| content | text | 活动内容（富文本） |
| type | tinyint | 活动类型 |
| status | tinyint | 状态（0草稿/1已发布/2已撤回/3已结束） |
| startTime | datetime | 开始时间 |
| endTime | datetime | 结束时间 |
| createBy | bigint | 创建人 |

### 3.2 活动角色推送

**数据库表**：sys_activity_role

**功能**：指定活动推送给哪些角色

**字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| activityId | bigint | 活动ID |
| roleId | bigint | 角色ID |

**业务规则**：
- 活动发布时根据角色推送
- 通过WebSocket实时通知
- 用户登录后查看属于自己的活动

**发布流程**：
```
1. 创建活动（草稿状态）
   ↓
2. 填写活动内容和时间
   ↓
3. 选择推送角色
   ↓
4. 发布活动
   ↓
5. WebSocket推送通知
   ↓
6. 用户收到通知
```

## 四、评价管理

### 4.1 评价功能

**功能描述**：用户对课程或考试进行评价

**数据库表**：sys_evaluation

**评价对象**：
| 类型编码 | 对象类型 | 说明 |
|---------|---------|------|
| 1 | 课程 | 课程评价 |
| 2 | 考试 | 考试评价 |

**接口**：
```
GET    /system/evaluation/page          # 分页查询评价
POST   /system/evaluation               # 提交评价
PUT    /system/evaluation/{id}/reply    # 回复评价
DELETE /system/evaluation/{id}          # 删除评价
PUT    /system/evaluation/{id}/status   # 审核评价
```

**评价字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| targetId | bigint | 评价对象ID |
| targetType | tinyint | 对象类型（1课程/2考试） |
| userId | bigint | 评价人 |
| score | tinyint | 评分（1-5星） |
| content | text | 评价内容 |
| reply | text | 回复内容 |
| status | tinyint | 状态（0待审核/1已通过/2已拒绝） |

**业务规则**：
- 学员完成课程/考试后才能评价
- 每个用户对同一对象只能评价一次
- 评价需要审核后才显示
- 管理员可回复评价

## 五、Excel模板管理（v2.3新增）

### 5.1 模板系统架构

**核心特性**：
- ✅ 可视化配置
- ✅ 在线预览
- ✅ 动态生成
- ✅ 样式自定义
- ✅ 数据验证
- ✅ 示例数据

**数据库表**：
- sys_excel_template：模板主表
- sys_excel_template_column：模板列配置表

### 5.2 模板主表

**字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| templateCode | varchar(100) | 模板编码（唯一） |
| templateName | varchar(200) | 模板名称 |
| templateType | tinyint | 类型（1导入/2导出） |
| businessModule | varchar(50) | 业务模块（question/course/exam） |
| sheetName | varchar(100) | Sheet名称 |
| headerRowHeight | int | 表头行高 |
| contentRowHeight | int | 内容行高 |
| defaultColumnWidth | int | 默认列宽 |
| headerBgColor | varchar(20) | 表头背景色（#4472C4） |
| headerFontColor | varchar(20) | 表头字体颜色（#FFFFFF） |
| status | tinyint | 状态（0禁用/1启用） |
| sortOrder | int | 排序 |

### 5.3 模板列配置

**字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| templateId | bigint | 模板ID |
| columnIndex | int | 列索引（从0开始） |
| fieldName | varchar(100) | 字段名 |
| columnTitle | varchar(200) | 列标题 |
| columnWidth | int | 列宽 |
| dataType | varchar(50) | 数据类型（String/Integer/Long/Double等） |
| required | tinyint | 是否必填 |
| visible | tinyint | 是否显示 |
| alignment | varchar(20) | 对齐方式（left/center/right） |
| exampleValue | varchar(500) | 示例数据 |
| dictCode | varchar(100) | 数据字典编码 |
| validationRules | text | 验证规则（JSON） |

### 5.4 初始化模板

**预置模板**：

#### question_import（题库导入模板）
- 业务模块：question
- 模板类型：导入
- 列数：10列
- 表头颜色：蓝色（#4472C4）

**列配置**：
| 列名 | 字段名 | 数据类型 | 必填 |
|------|--------|---------|------|
| 题目内容 | questionTitle | String | 是 |
| 题型 | questionType | Integer | 是 |
| 选项 | options | String | 否 |
| 正确答案 | correctAnswer | String | 是 |
| 学科ID | subjectId | Long | 否 |
| 年级 | grade | String | 否 |
| 难度 | difficulty | Integer | 否 |
| 分值 | score | Double | 否 |
| 知识点 | knowledgePoint | String | 否 |
| 解析 | analysis | String | 否 |

#### course_export（课程导出模板）
- 业务模块：course
- 模板类型：导出
- 列数：11列
- 表头颜色：绿色（#70AD47）

### 5.5 模板接口

```
GET    /system/excel-template/page           # 模板分页查询
GET    /system/excel-template/{id}           # 模板详情
POST   /system/excel-template                # 新增模板
PUT    /system/excel-template                # 修改模板
DELETE /system/excel-template/{id}           # 删除模板
GET    /system/excel-template/{id}/preview   # 预览模板
GET    /system/excel-template/download/{code} # 下载模板
POST   /system/excel-template/export/{code}  # 导出数据
```

**使用场景**：
- 题库批量导入
- 课程数据导出
- 考试成绩导出
- 学员信息导出

## 十、业务规则

1. **配置缓存**：系统配置加载后缓存，修改后刷新
2. **活动推送**：根据角色精准推送
3. **评价审核**：评价需审核后显示
4. **邮件日志**：所有邮件发送记录日志
5. **模板复用**：Excel模板可跨业务复用
6. **家长权限**：家长仅查看学生学习数据
7. **单词训练**：支持随机抽取、批量导入
8. **数据验证**：Excel导入时校验数据格式
